# SachoolBot

A multi-purpose discord bot made to help students with keeping track of their assignments.

## Description

SachoolBot is a discord bot that helps students with their assignments. It contains many useful built-in utilities for helping students with their work.
Some of these features include:

- Schedule management
- Assignment deadline management
- File conversion
- LLM integration
- Text-to-speech for pronounciation
- AI voice call integration for easy study access.
- Integrated web searching
- and more!
